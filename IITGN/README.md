# 🛡️ IIT Gandhinagar

## Cybersecurity & Threat Intelligence Intern
**Guide:** Prof. Sameer Kulkarni  
**Duration:** June 2025 – October 2025

---

## 📌 Overview

Worked on applied machine learning and threat intelligence techniques for phishing detection and malicious URL analysis.

The internship focused on designing an explainable and scalable framework for identifying malicious websites through multiple sources of evidence and performing intelligent threat assessment in real time.

---

## 🎯 Problem Statement

Malicious websites frequently employ phishing, typosquatting, and deceptive domain naming techniques to target users and evade traditional security mechanisms.

The objective was to design a multi-layered and explainable URL analysis framework capable of integrating diverse sources of evidence to improve malicious website detection.

## 🚀 Major Outcome

Designed **AURIS (Automated URL Risk Intelligence System)**, an explainable multi-layered URL security framework integrating:

- Threat Intelligence
- DNS and WHOIS Analysis
- SSL Validation
- Fuzzy Matching
- Reputation Analysis
- Admiralty-style Risk Scoring

### Evaluation Results

- 90.5% Classification Accuracy
- 92.2% Specificity
- 87.2% Recall
- Evaluated across 1,056 real-world URLs
