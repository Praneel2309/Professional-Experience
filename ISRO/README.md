## 📌 Overview

Worked on machine learning approaches for cybersecurity applications, with a focus on Domain Generation Algorithms (DGAs) and malicious domain detection.

The internship involved investigating how intelligent systems can distinguish algorithmically generated malicious domains from legitimate internet traffic using data-driven techniques.

---

## 🎯 Problem Statement

Many modern malware families employ Domain Generation Algorithms (DGAs) to continuously generate new domains and evade traditional blacklist-based security systems.

The objective was to explore intelligent techniques capable of identifying malicious domains with high reliability and scalability.
